Manifest V3

Bootcamp Helper — Extensão MV3

Extensão para Chrome (Manifest V3) — projeto do Bootcamp II.

Instalação manual (modo desenvolvedor)

  1. Acesse chrome://extensions e ative Developer mode.
  2. Clique em Load unpacked e selecione a pasta do projeto.
  3. Abra o popup pelo ícone da extensão e verifique o console (F12).

Download

Baixar .zip

Caso não exista, crie uma Release no GitHub com o zip.

Permissões

  • storage
  • Content script apenas em developer.chrome.com

Como funciona

O popup envia um PING para o service worker, que responde com o horário. Um content script simples destaca links em páginas do domínio permitido.

Veja o código no GitHub e a documentação do MV3.
